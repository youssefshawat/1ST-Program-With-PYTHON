🚀 STL Volume & Mass Analytics ToolA powerful, easy-to-use command-line tool designed for 3D printing enthusiasts and engineers to calculate volume, surface area, bounding box, and estimated mass of 3D models. Supports STL (ASCII/Binary), NIfTI, and DICOM formats.✨ Key FeaturesDeep Analysis by Default: Get instant file size, triangle count, bounding box, surface area, and volume with a single command.Smart Infill Comparison: Compare model mass at custom infill percentages (default 20%) vs. a 100% solid version.Multi-Material Estimation: Instantly calculates estimated mass for over 20+ professional materials (PLA, ABS, Titanium, Gold, etc.).Pro Console UI: Data is presented in beautifully formatted, easy-to-read tables.Developer Friendly: Full JSON output support for seamless integration into your own scripts or web apps.Medical Format Support: Beyond standard STL, it handles NIfTI and DICOM files.🛠️ InstallationEnsure you have Python 3.8+ installed. You can install the tool directly from the source:Clone the repository:Bashgit clone https://github.com/youssefshawat/STL-Volume-Analytics.git
cd STL-Volume-Analytics
Install the package:Bashpip install .
🚀 UsageAfter installation, run the volume-calculator command from any directory.Full Model AnalysisBashvolume-calculator MyModel.stl
Custom Infill & UnitBashvolume-calculator MyModel.stl --infill 15 --unit cm --output-format table
📊 Command-Line ArgumentsArgumentDescriptionfilenamePath to your model file (STL, NIfTI, DICOM).--calculationOptimize by running: volume or area.--infill <%>Infill percentage (Default: 20.0).--unit <unit>Display in cm (default) or inch.--output-formatChoose table (default) or json.--list-materialsShow all supported materials and IDs.🧪 Supported MaterialsIncludes density profiles for:Polymers: PLA, ABS, PETG, Resin, Nylon.Metals: Aluminum, Steel, Titanium, Brass, Copper, Gold (14K/18K).Specialty: Alumide, Carbon Fiber (3k CFRP), Red Oak.👨‍💻 AuthorYoussef Yasser (Youssef Shawat)Data Science & AI Student | Automation ExpertGitHub: github.com/youssefshawatLinkedIn: [www.linkedin.com/inyoussef-shawat-49971939a]📝 LicenseThis project is licensed under the MIT License - see the LICENSE file for details.
